Anchor link grid

A grid of button links, usually located in the hero banner or near the top of the page, that anchor link to different sections of the same page.

Component features and rules:

  • They should always have an icon and text to set them apart from regular button links.
  • Text should always be left-aligned, with all text and icons vertically aligned center.
  • Button heights should be stretched to the tallest item.
  • Subtle borders on the buttons give them seperation when stacked.
  • There is no standard width for the buttons, as that should be determined by the designer/coder when adding the grid, depending on the number of links and length of text inside each one.
  • There is a light and dark version for the icon and text colors and hover states, but the background color for all the buttons should be determined seperately with a background-[color] class.
