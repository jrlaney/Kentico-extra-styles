Advanced Analytics

Sophisticated analytics made easy.

Maximize the impact of your business applications with powerful, advanced, and predictive analytics. MicroStrategy provides an extensive library of native analytical functions and scoring algorithms, along with an SDK to integrate with 3rd-party and open source statistical and data mining products.

Watch webcast

Your data can help predict the future of your business.

Harness historical data to make better, more informed decisions with data mining and predictive analysis delivered through a fully-integrated, enterprise-caliber BI platform. Create real-time forecasts by leveraging advanced, native analytical capabilities, with the option to easily incorporate third-party data mining and modeling tools. MicroStrategy Data Mining Services allows users to build and share predictive reports viewable on any device.

300+ native analytics functions that cover the entire analytical spectrum.

MicroStrategy comes with an extensive library of over 300 OLAP, mathematical, financial, and data mining functions that can be used to better understand the relationships between data, create business metrics and top-level KPIs, or build advanced statistical analyses. Organizations can leverage these functions across their entire MicroStrategy environment and in any medium from reports and statements, to dashboards, to enterprise mobile applications.

Extend out-of-the-box functionality by integrating with open source and 3rd-party statistical programs.

R analytics is a leading open-sourced, 3rd-party statistical software. MicroStrategy offers an innovative integration with R that’s fast, simple, and scalable. In just a few clicks, any algorithm, statistic, or analysis generated in R can be seamlessly incorporated into a MicroStrategy report, dashboard, or visualization.

Predictive Model Markup Language (PMML) is a standardized language for data mining models developed by the Data Mining Group (DMG). It’s the output generated by many leading data mining tools including, SAS®, SPSS®, Microsoft®, Oracle®, and IBM®. MicroStrategy is able to automatically convert PMML outputs into an algorithm that can be used by metrics in MicroStrategy, so you can leverage the work done in these data mining tools directly in your BI applications.
