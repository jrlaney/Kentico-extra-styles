Simple, Fast Deployment

With just a few clicks, organizations can deploy fully configured enterprise environments. Most instances can be up and running in just 30 minutes. Flexible and Easy to Maintain

Customers that deploy a MicroStrategy project on the cloud can dynamically start, stop, restart, and resize their environments. Organizations can reduce costs by stopping their environments during non-peak hours. Easily monitor and manage the environment with real-time notifications, automation, and scheduling.

Scalability

Seamlessly scale the environment up or down based on usage, without a long-term commitment. Customers can choose from 6 different instance size options ranging from 4 vCPU and 15.25 GiB to 64 vCPU and 488 GiB of memory. Organizations can add new users at any time, deploy multiple environments and projects as they see fit, and pay only for what they use.

Global Availability

With MicroStrategy on the cloud, benefit from distributed and redundant systems in 42 availability zones covering 16 geographic regions around the world. Deploying closer to your user base reduces latency and allows for faster response times. And using regional data centers helps organizations ensure compliance with local laws and regulations.

Security and Compliance

MicroStrategy on AWS maintains compliance with an industry-accepted risk management/privacy framework that includes:

  • Service Organization Controls (SOC 2 Type II)
  • Health Insurance Portability and Accountability Act of 1996 (HIPAA)
  • Payment Card Industry Data Security Standard (PCI DSS)
  • Privacy Shield
  • International Organization for Standardization (ISO 27001)
  • General Data Protection Regulation (GDPR)

The system also conducts weekly vulnerability scans and remediation activities, completes penetration testing to proactively assess systems by simulating attacks, and maintains a record of changes to the MicroStrategy application by tracking changes to objects.

Platform Adminstration

The platform administration features include:

  • Application KPIs that are collected every five minutes. Alerts are sent to a 24/7 MicroStrategy emergency response team
  • Scheduled windows for application upgrades and maintenance
  • Daily backups of cubes, caches, images, customizations, metadata, Enterprise Manager, and mobile configuration
  • Standard authentication and Usher authentication support for all MicroStrategy applications. This includes external authentication such as LDAP and SSO
  • MicroStrategy over-the-top application hotfix and patch installation
  • ODBC.INI configuration support and connectivity verification to customer warehouse, including VPN, security groups, and direct connect configuration and maintenance

Environment and Operating System

The MicroStrategy on AWS environment features:

  • Rapid deployment of MicroStrategy on AWS infrastructure, using best practices within a customer-specific VPC
  • Infrastructure system KPIs, which are collected every five minutes. Alerts are sent to a 24/7 MicroStrategy emergency response team
  • Scheduled maintenance windows for system patching, security updates, and AWS infrastructure management
  • Daily backups of metadata, operating system files, and customer data
  • A quarterly service review that captures major configurations of the application, infrastructure performance, support case snapshots and statistics, and other KPIs
    •  
  • VPN tunnel and/or direct connect to your warehouse
  • Disaster recovery testing to ensure reliability and stability

Services

All MicroStrategy managed Cloud environments receive the following services:

  • Architected to operate across multiple Availability Zones
  • Dedicated platform within your own Virtual Private Cloud (VPC)
  • Highest level of technical support within Amazon (Enhanced Support)
  • Multiple Amazon Technical Account Managers
  • Elastic IP addresses assigned to your environment
  • Flexible storage options of Elastic File System (EFS) or Elastic Block Storage (EBS)
  • Relational Database Service deployed in multi-availability zones with general purpose SSD drives and automatic patching services, automated backups, and automatic host Failover
  • Monitoring and alerting capabilities via the CloudWatch Service
  • Certificate Manager service that lets you easily provision, manage, and deploy Secure Sockets Layer/Transport Layer Security (SSL/TLS) certificates
  • Identity and access management (IAM) to control users' access to AWS services. Create and manage users and groups, and grant or deny access
  • Amazon GuardDuty threat detection that continuously monitors and protects your AWS environment
