MicroStrategy on AWS delivers instant access to a complete enterprise cloud platform. Organizations of any size can now launch fully configured and ready-to-use enterprise analytics and mobility projects, completely on their own. Administrators, for the first time, have total control of their environments, making it easy to start, stop, resize, and terminate as needed – ensuring access to the latest features and capabilities while delivering the lowest possible TCO.

Comprehensive Capabilities

MicroStrategy delivers the full breadth of analytical capabilities, including: enterprise data discovery, enterprise reporting and dashboards, optimized data connectivity, personalized distribution capabilities, and advanced predictive analytics.

Powerful Mobile Functionality

Mobile analytics and productivity apps, customized with transaction-enabled workflows, can be tailored to meet the needs of any business function or role.

High-performance Architecture

Regardless of whether you deploy in the cloud or on-prem, the MicroStrategy platform is uniquely suited to deliver high performance applications that meet the business intelligence demands of any organization. Learn how to quickly deploy sophisticated analytics and mobility applications at scale.

Why MicroStrategy on AWS?

We've worked with Amazon Web Services to take care of the hard stuff, so you get access to an amazing analytics solution without sweating the details. With best-in-class security, performance, reliability, and support, your analytics is in good hands.

Simplified Enterprise Deployment

With just a few clicks, organizations can deploy fully configured enterprise environments. Most instances can be up and running in just 30 minutes. This saves users time and money by eliminating the need to download, install, and configure environments themselves. Users can choose from different instance types and sizes based on their budget and usage requirements.

Centralized Administration

Customers that deploy a project using MicroStrategy on AWS can dynamically start, stop, restart, and resize their environments. Organizations can reduce costs by stopping their environments during non-peak hours.

Scalability

Scaling your MicroStrategy on AWS environment is quick and easy. Customers can choose from 6 different instance size options ranging from 4 vCPU and 15.25 GiB to 64 vCPU and 488 GiB of memory. Environments can be scaled on-demand or on a preset schedule based on usage patterns or data size. Organizations can add new users at any time, deploy multiple environments and projects as they see fit, and pay only for what they use.

API Enabled Operations

The MicroStrategy Cloud API provides unlimited flexibility for organizations to create customized workflows. Developers can build personalized user interfaces for deploying, managing, and automating their cloud environments. It’s also easy to integrate with other automation tools, and the API works with any programming language. With the API it is possible to launch multiple MicroStrategy instances with a single line of code, resize them as needed, and terminate them when they are no longer required.

Unrivaled Cloud Security

Backed by the cutting-edge security functionality of Usher, MicroStrategy meets or exceeds leading industry security standards and best practices.

Global Availability

With MicroStrategy on AWS, organizations can deploy distributed and redundant systems in 42 AWS Availability Zones covering 16 geographic regions around the world (including North America, South America, Asia, Europe, and Australia). Deploying closer to your user base reduces latency and allows for faster response times. And using regional data centers helps organizations ensure compliance with local laws and regulations.
